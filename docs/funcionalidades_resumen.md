# Resumen de Funcionalidades

Este archivo resume, en lenguaje no técnico, las funciones principales de la aplicación actual.

- Crear modelos 3D simples dibujando líneas y puntos.
- Formar caras automáticamente cuando el contorno queda cerrado.
- Colocar figuras del inventario como cubos, esferas, cilindros, pirámides o conos.
- Cambiar entre varios modos de control: líneas, bloques con teclado, bloques con ratón y selección de cara.
- Usar flechas o `WASD` para construir con teclado.
- Seleccionar puntos, bloques o caras con el ratón para editarlos.
- Limpiar automáticamente la selección múltiple de puntos al hacer clic izquierdo simple fuera del trabajo actual.
- Borrar puntos o bloques seleccionados con `Delete` o retroceso.
- Dividir cubos en piezas más pequeñas y seguir editándolas.
- Fusionar bloques completos o solo la selección actual.
- Aplicar texturas a caras concretas o a todo el modelo.
- Recolocar las UV en una ventana modal de edición visual.
- Cargar y gestionar una biblioteca pequeña de texturas dentro del proyecto con una lista amplia de formatos de imagen.
- Ver y usar un acceso rápido de texturas en la esquina inferior izquierda.
- Guardar el proyecto completo del editor y volver a cargarlo más adelante para seguir trabajando.
- Importar modelos `OBJ` o `FBX` desde el menú `Archivo`.
- Exportar el trabajo como `GLB` o `OBJ`.
- Deshacer y rehacer cambios con atajos compatibles con Windows y Mac.
- Mostrar u ocultar los FPS desde el menú `Ver`.
- Mostrar una ayuda visual compacta con emojis para recordar los controles.
- Ejecutar la app como ventana de escritorio con menús nativos del sistema.
