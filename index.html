Gestor de Texturas y UV

Carga una textura y recoloca la malla UV dentro del cuadrado, como en un editor UV tipo Blender.

Espacio UV

El cuadrado representa toda la textura. La malla naranja es la parte del modelo que estás editando.

U
Editor UV listo Carga una textura y elige si quieres editar una parte seleccionada o toda la malla del modelo.

Arrastra dentro del recuadro para mover la textura sobre la malla. Tira de una esquina para ampliar o comprimir el encaje.

Alcance

Usa "Seleccionar cara" si quieres texturizar solo una parte concreta. Si eliges "Modelo completo", la textura y el editor UV actuarán sobre toda la malla disponible.

Tipo de seleccion conjunta

Tipo 1 usa la seleccion conectada anterior. Tipo 2 limita la seleccion a una superficie rectangular continua.

Consejo: si quieres imitar Blender, carga una textura, selecciona una cara o todo el modelo y después recoloca la isla UV en el cuadrado.
